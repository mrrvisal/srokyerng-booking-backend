# Backend Docs

Use this folder for team-facing backend decisions and workflows.

## Files

- `setup-guide.md`: local setup, environment variables, database setup, and run commands
- `api-contract.md`: current endpoint contract, roles, auth header, and response shape
- `backend-rules.md`: coding rules for modules, routes, services, models, validation, auth, and errors
- `module-ownership.md`: module ownership, scaffold policy, and team responsibilities

Keep backend decisions documented here when they affect more than one teammate.
